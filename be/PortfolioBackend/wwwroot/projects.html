Experience

Software Engineering Internship

NI (National Instruments)

May 2023 - August 2023

Overview

My Internship was what got me started in getting industry skills relating to software Engineering. I worked with a team of professional software engineers to develop a desktop application for one of the company's testing software. I was able to learn more about desktop development using C# and WPF, as well as gaining experience in working with version control and participating in code reviews as the project was constantly evolving. The internship period also gave me experience in working with a team of professional software engineers and exposure to practices within a company. Although this was a project that I worked on independently, I had a mentor and supervisor that I would meet with on a weekly basis to discuss my progress and any blockers that I had, as well as any other topic relating to Software Engineering and the company. The internship was within a program that let me meet other interns and get to their projects and experiences as well.

Impact

I was able to surpass expectations by being able to move past the original scope of the project, which was to develop a prototype. Past this, I was able to develop unit tests and began the implementation of environment files and UI before reaching the end of my internship. I was able to meet one of the company's product managers relating to my application and other teams in order to give an application demo and gain feedback in order to continue development. I was also able to document my progress relating to what the application does and the research I needed to do in order to make it work. These documents were shared on the cloud where my team would be able to read and pick up where I left off on the project. The git practices also left the project in a state where it could be picked up by another developer and continue development.

Skills learned

  • C# and WPF
  • .NET Ecosystem
  • Visual Studio
  • Version Control
  • Collaboration with a professional team
  • Code reviewing
  • Experience in Agile Practices
  • Documentation for developing applications
